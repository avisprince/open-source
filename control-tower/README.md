## Installation

```bash
$ yarn install
```

## Setup ENV

```bash
Copy the file .env.template into a new file .env.local
Replace all values of <replaceme>
```

## Running the app

```bash
$ yarn start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
